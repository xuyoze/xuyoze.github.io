Python3.7下安装scrapy报错解决犯法

12月 20, 2018 发布在 转载

升级python 到3.7后, 安装scrapy 遇到报错情况, 网上查询看到如下文章 转载记录

在python3.6以上版本安装scrapy框架是会报错缺少Microsoft Visual C++ Build Tools

然后安装会出现下面错误

2.去www.lfd.uci.edu 下载twisted对应版本的whl文件，这里对应版本是只对应你安装的python的版本，寻找你所需要的版本下载下来

3.pip install 你下载的whl文件

4.最后pip install scrapy即可
